Display Inline vs Inline Blocks

10/23/14

When writing the CSS to complement your HTML code, you may find that you want certain elements to display their content in rows and columns. Luckily, there is a way to do this easily with the display attribute. The display attribute has 3 different options you can choose from; block, inline and inline-block.

Take a look at the image below. The following image has a container width of 50px.

  • Display: block will display each content as if it were blocks going down the container.

  • Display: inline will display the content across the width of the container and wraps the content.

  • Display: inline-block will display the content as a block in a line across the width of the container. It will wrap the content around confining it to its content block size.
