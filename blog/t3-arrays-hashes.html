Arrays and Hashes

10/24/14

What is an array?

First guess.. Communications Array like in video games? Well yes, a communications array is exactly that but we're talking about an array in programming. More like this.

sample_array = [ 0, 1, 2, 3, "i am a string inside an array"]

As you can see, sample_array is created by specifying the elements that will exist inside of the array container. Once these array elements are assigned, you can reach them by typing the array name followed by [x], where x is the index of the array.

Reading the index of the array can get a little bit tricky so you have to be careful. The first element within the array is always index 0. Then the rest will proceed in numerical order. Here's an example. Here you have an array with 5 elements. The element content 2 has the index of [0]. The element 5 has the index of [1]. The element 1 has the index of [2] and so on.

If you want to add or remove from the array somewhere along your script, you have the options of .push and .pop.

  • push: This command will add the element of your choice to the end of your array.
  • pop: This command will remove the last element in your array entirely. Kaboom!

    What is a hash? A hash is a collection of unique keys associated to their corresponding values. Each key will have a value associated with it. Think of it like logins and passwords. The login is associated to its password via Hash. Here's an example of a hash.

    sample_hash = [ :movies => "avengers2" ] You can also create a Hash using its ::new method. For example, sample_hash = Hash.new. Then you can start associate keys with new values. Then type sample_hash["blog_number"]=5
