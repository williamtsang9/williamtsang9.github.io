Javascript vs Ruby

11/29/14


What's the difference?

Javascript and Ruby have MANY similarities and differences. I'd like to start with saying Ruby is probably still my favorite language. While Javascript is extremely powerful and works quite well with HTML, Ruby has so many existing methods and enumerables available making the language extremely simple and easy to use. This is just my opinion though.

Similarities

You know, when we first worked on Ruby arrays and hashes, I thought OK I can get used to this. I got familiar with it and then came Javascript. It came to my attention that JavaScript objects closely resemble Ruby Hashes! Mind blown. I guess languages share core similarities that differ only by name.
